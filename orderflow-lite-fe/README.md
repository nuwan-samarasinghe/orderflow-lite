# orderflow-lite-fe

Angular frontend for the OrderFlow Lite interview showcase.

## Highlights

- Standalone components
- Lazy loaded routes
- Angular Material UI
- Signals for local and derived UI state
- RxJS for async search and API flow
- NgRx for shared products state
- HTTP interceptors for correlation id and API error handling
- OnPush change detection
- Strict TypeScript configuration

## Run

```bash
npm install
npm start
```

## Build

```bash
npm run build
```
